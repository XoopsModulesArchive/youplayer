VIDEO EMBED INFO:
Sie können diese mit einigen embed code ähnlich wie die von YouTube verwendet einbinden,
 oder Sie können SWFObject verwenden (wie Sie sehen in der heruntergeladenen Dateien)
Es gibt 3 Variablen
Playlist = Ort den Namen eines XML-Playlist (ohne. xml am Ende)
autoplay = true oder false
rollover_changing_thumbs = true oder false (falls Sie die Thumbnails, um on-roll over ändern)


XML-Struktur:
Wie Sie sehen, ist die XML-Struktur sehr einfach
Wann immer Sie wollen ein neues YouTube-Video, 
das Sie brauchen nur auf einen Knoten mit dem Namen 
"Titel", einen Knoten mit dem Namen "ID" und einen Knoten mit dem Namen "Beschreibung"
Du hast hier ein Beispiel:
<track>
    <title> Die YouTube-Video </ title>
    <youtube> YOUTUBE_ID </ youtube>
    <detail> Beschreibung (unterstützt HTML-Text mit CDATA) </ Details>
</ Titel>
